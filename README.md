## 🚀 Deployment Options

### Option 1: Netlify (Recommended — Free)
1. Go to [netlify.com](https://netlify.com) → Sign up free
2. Drag and drop this entire folder onto the Netlify dashboard
3. Your app is live in 30 seconds at a `*.netlify.app` URL
4. Optional: Add your custom domain

### Option 2: Vercel (Free)
1. Go to [vercel.com](https://vercel.com) → Sign up free
2. Click **Add New → Project → Upload folder**
3. Upload this folder → Deploy

### Option 3: GitHub Pages (Free)
1. Create a GitHub account at [github.com](https://github.com)
2. Create a new repository (e.g. `dheeeran-tractors`)
3. Upload all files from this folder
4. Go to **Settings → Pages → Source: main branch**
5. Your app is live at `https://yourusername.github.io/dheeeran-tractors`

### Option 4: Any Web Host (cPanel / Hostinger / GoDaddy)
1. Open your hosting File Manager
2. Navigate to `public_html` folder
3. Upload all files from this folder
4. Visit your domain — done!

### Option 5: Local / Offline
Just open `index.html` in any browser — works completely offline.

## 📱 Install as Mobile App (PWA)

Once deployed to a URL:
1. Open the URL in **Chrome** (Android) or **Safari** (iPhone)
2. Tap **"Add to Home Screen"**
3. The app installs like a native app with an icon

## ✨ Features

- **Splash Screen** with animated logo
- **3-screen Onboarding** flow
- **OTP Authentication** (phone number)
- **Dashboard** with live stats & commission tracker
- **Add/Edit Tractor** — 7 collapsible sections with smart dropdowns, sliders, chip selectors, photo upload
- **Listings Page** — search, filter, sort
- **Tractor Detail** — full specs, gallery, actions
- **Share Screen** — branded poster, WhatsApp direct share, Instagram caption, PDF
- **Reports** — monthly charts, export to Excel/PDF
- **Custom Fields** builder
- **Settings** — profile, branding, contact, toggles
- **LocalStorage** persistence — data survives page refresh
- **PWA ready** — installable on mobile
